# SGTM - Sistema de Gestión de Transporte Marítimo Buenaventura

Proyecto para gestionar operaciones portuarias (tareas, embarques, embarcaciones, personal, facturación, estadísticas y exportación de datos).

---

## Funcionalidades principales

- **Gestión de tareas:** creación, visualización y control de actividades operativas.

- **Administración de embarques y embarcaciones:** registro y seguimiento de barcos y cargas.

- **Gestión de personal:** manejo de la información del personal involucrado en las operaciones portuarias.

- **Módulo de facturación:** control básico de cobros y pagos asociados a las operaciones.

- **Estadísticas:** visualización de datos mediante gráficos e indicadores.

- **Exportación de datos:** permite descargar información en diferentes formatos para análisis externo.

- **Interfaz modular**: cada sección del sistema funciona de forma independiente, permitiendo una fácil ampliación o mantenimiento.

---

## Tecnologías utilizadas

### Frontend

- **HTML5** : estructura principal del proyecto.

- **CSS3 + Tailwind CSS:** diseño responsivo, limpio y moderno con utilidades predefinidas.

- **JavaScript**: manejo de la lógica de negocio, eventos, y comunicación entre módulos.
